# 🚀 SICHS - DTCEA-SRO

### Sistema Integrado de Controle de Hidrometria e Serviços

![Angular](https://img.shields.io/badge/Angular-21-DD0031?style=for-the-badge&logo=angular)
![Laravel](https://img.shields.io/badge/Laravel-11-FF2D20?style=for-the-badge&logo=laravel)
![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)

O **SICHS** é uma solução web desenvolvida para centralizar e otimizar a gestão de hidrometria e o controle operacional do **DTCEA-SRO**. O sistema automatiza fluxos de leitura, cálculos de consumo e gestão de pessoal em uma interface única e moderna.

---

## 🔍 Visão Geral

O sistema foi projetado para substituir processos manuais, centralizando os principais fluxos de operação do setor. Através de um dashboard responsivo, os gestores podem monitorar o consumo de medidores e manter o cadastro do efetivo militar atualizado em tempo real.

---

## ✨ Funcionalidades

- **🚰 Leituras de Hidrômetros**:
  - Cadastro e consulta simplificada.
  - Cálculo automático de consumo entre períodos.
  - Histórico de registros com opção de edição e exclusão.
- **🪖 Gestão de Militares**:
  - CRUD completo para manutenção de dados cadastrais.
  - Controle de acesso e organização do efetivo.
- **📊 Relatórios Profissionais**:
  - Geração de documentos em PDF via **DomPDF**.
  - Filtros personalizados por medidor e intervalo de datas.
- **🖥️ Interface Moderna**:
  - Dashboard responsivo para acesso via desktop ou dispositivos móveis.
  - Experiência de usuário (UX) consistente e intuitiva.
